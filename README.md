### App Installation

```
npm create vite@latest my-app -- --template react
```

### App run

```
npm install
npm run dev
```

### Used technologies

- React.js
- Redux
- Tailwind CSS

### Features

- OTP based login with Mobile Number
- Logged-in user can access user creation, upload file, file search and preview

### File structures

- pages
- components
- services
- redux
